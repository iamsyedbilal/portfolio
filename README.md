# Syed Bilal — Portfolio

Personal portfolio website built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

**Live:** [syedbilal.vercel.app](https://syedbilal.vercel.app)

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion

## Sections

- Hero
- About — skills, stats, currently
- Projects — featured and noteworthy
- Journey — timeline
- Contact

## Run Locally

```bash
git clone https://github.com/iamsyedbilal/portfolio
cd portfolio
npm install
npm run dev
```
